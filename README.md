
# 4Fs of Weight Loss Portal

Helps members track their progress through various aspects of the 4Fs of Weight Loss program.


## Tech Stack

**Client:** React, Context API, Styled Components, Semantic UI

**Server:** Node, Express

**Database:** PostgreSQL
